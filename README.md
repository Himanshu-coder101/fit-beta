# FitTrack Pro — Coach-like plans (Prototype)

This version implements a professional onboarding flow:
- Goal → Experience → Time → Frequency → Equipment → Style
- Deterministic plan generator producing exercises with sets/reps/intensity
- Per-day pages showing exercises and feedback
- LocalStorage demo for profile; API route returns the generated plan

Run:
1. npm install
2. npm run dev
3. Open http://localhost:3000

This is a prototype for demonstration and development. For production add authentication and a database.
